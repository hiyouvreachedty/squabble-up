# Squabble-Up 🥊

> Boxing studio & personal portfolio website — conversion-first, brand-forward, built with Next.js + TypeScript.

---

## Overview

Squabble-Up is a hybrid boxing studio and creative portfolio website for Terrell Pelletier, based in Kelowna, BC. Inspired by the high-energy, premium feel of boutique fitness brands, but built with a personal, design-forward identity — not a corporate franchise template.

The site serves two primary goals:
1. **Studio** — showcase classes, training philosophy, booking, and location
2. **Portfolio** — highlight design work, development projects, and creative direction

---

## Tech Stack

| Layer | Tool |
|---|---|
| Framework | Next.js 14 (App Router) |
| Language | TypeScript |
| Styling | Tailwind CSS |
| Animation | Framer Motion |
| Deployment | Vercel |
| CMS (future) | Sanity or Contentlayer |
